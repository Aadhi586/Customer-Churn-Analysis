# 📊 Customer Churn Analysis and Prediction

A real-world Data Analysis project developed as part of my **Data Analysis Internship at SaiKet Systems**. This project focuses on analyzing customer churn behavior in a telecommunications company and building a machine learning-ready dataset for predictive analysis.

## 🚀 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. This project aims to explore customer data, identify factors influencing churn, and prepare the dataset for visualization and predictive modeling.

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Handle missing values and duplicates
- Convert data types for analysis
- Encode categorical variables
- Explore customer behavior through EDA
- Segment customers based on key features
- Build a churn prediction model using Machine Learning
